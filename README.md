# Sentiment Analysis on Tweets using NLP and Machine Learning

## Overview

This project implements a Sentiment Analysis system that classifies tweets into Positive, Negative, or Neutral sentiments using Natural Language Processing (NLP) and Machine Learning techniques.

The project demonstrates the complete NLP workflow including text preprocessing, feature extraction, model training, evaluation, and prediction on custom user input.

## Features

- Text preprocessing and cleaning
- Lowercase conversion
- Number removal
- Punctuation removal
- Emoji removal
- URL removal
- Stopword removal
- Bag of Words (BoW) Vectorization
- TF-IDF Vectorization
- Logistic Regression Classification
- Performance Evaluation using Precision, Recall, F1-Score
- Custom User Input Prediction


## Machine Learning Workflow

1. Load Tweet Dataset
2. Clean and preprocess text data
3. Convert text into numerical features using:
   - Bag of Words
   - TF-IDF
4. Split dataset into training and testing sets
5. Train Logistic Regression model
6. Evaluate model performance
7. Predict sentiment for new user-provided text
